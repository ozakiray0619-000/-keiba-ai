# 精度を上げる手順（Cursorのターミナルで実行）

netkeibaへのアクセスはAI側の環境からは遮断されているため、
データ収集はこのPC（Cursor）で実行してください。流れは「①集める → ②学習 → ③確認」。

## 0. 準備（最初の1回だけ）
```
cd keiba_app
venv\Scripts\activate          # Windows
pip install -r requirements.txt
```

## 1. 過去レースを集める（最重要）
今は74レースしかありません。まず90日分を試しましょう。
```
python -m pipeline.collect_historical --days 90
```
- 1リクエスト2〜3秒待つので90日で数十分〜。途中で止めても再開可（取得済みはスキップ）。
- もっと増やすなら日付範囲指定:
```
python -m pipeline.collect_historical --from 2024-01-01 --to 2025-05-01
```

## 2. AIを再学習
```
python model/train.py
```
最後に出る `OOF AUC` が賢さの目安（0.50=カン、1.0=完璧）。
データが増えるほどこの数字が上がります。

## 3. 予測の答え合わせ（精度の記録）
レース結果が出たら `data/manual_predictions.json` の
`actual_result.finish_order` に着順（馬番）を記入して:
```
python model/score_predictions.py
```
券種ごとの的中率（通算）が出ます。
